# UX Review Dimensions

10 dimensions for comprehensive UX evaluation.

## Dimensions

| Dimension | Focus | Key Questions |
|-----------|-------|---------------|
| **Friction** | User journey obstacles | Steps to goal? Where do users hesitate/abandon? |
| **Delight** | Emotional design | Celebrations? Personality? Micro-interactions? |
| **Accessibility** | WCAG compliance | VoiceOver? Dynamic Type? 44pt touch targets? |
| **Monetization** | Revenue vs UX balance | Paywall timing? Value communicated? Dark patterns? |
| **Retention** | Habit formation | Return triggers? Streaks? Progress visibility? |
| **Virality** | Share mechanics | Share friction? Viral loops? Social proof? |
| **Onboarding** | First-run experience | Time-to-value? Permission timing? Progressive disclosure? |
| **Trust** | Safety signals | Permission context? Privacy transparency? Error handling? |
| **Cognitive Load** | Mental burden | Information density? Decision complexity? Memory requirements? |
| **Platform** | HIG compliance | Native patterns? Standard components? Safe areas? |

## Scoring

- **10**: Exceptional, best-in-class
- **7-9**: Good, minor improvements possible
- **4-6**: Adequate, notable issues
- **1-3**: Poor, significant problems
- **0**: Missing/broken
