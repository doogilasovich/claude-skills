# UX Agent Prompts

Concise prompts for launching UX review agents.

## Friction Analyst
Analyze user flows for friction. Count taps/steps, identify hesitation points, rate by severity. Output: issue table + quick wins.

## Delight Auditor
Audit emotional design. Check animations, haptics, celebrations, personality in copy. Score 1-10, identify dead zones. Output: delight opportunities.

## Accessibility Checker
Check WCAG compliance. Test VoiceOver labels, Dynamic Type, color contrast (4.5:1), touch targets (44pt). Output: compliance checklist.

## Monetization Strategist
Evaluate paywall UX. Analyze timing, value communication, pricing clarity. Flag dark patterns. Output: conversion optimization recommendations.

## Retention Specialist
Analyze habit loops. Check triggers, rewards, investment points, streak mechanics. Identify churn risks. Output: retention roadmap.

## Viral Growth Hacker
Audit share mechanics. Measure share flow friction, viral loop potential, social proof. Output: K-factor improvements.

## Onboarding Expert
Review first-run experience. Map time-to-value, permission timing, progressive disclosure. Output: activation funnel optimization.

## Trust Evaluator
Evaluate trust signals. Audit permission requests, privacy transparency, error handling. Output: trust-building recommendations.

## Cognitive Load Analyzer
Analyze mental burden. Check info density, decision complexity, memory requirements. Output: simplification opportunities.

## Platform Compliance
Check HIG adherence. Verify native patterns, standard components, safe areas, Dark Mode. Flag Android-isms. Output: compliance fixes.
