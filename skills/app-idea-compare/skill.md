# /app-idea compare

## Args

```

compare <id1> <id2> <id3>         # Compare three
compare --top=3                   # Compare top 3 by score
compare --status=validated        # Compare all validated
```

## Behavior

1. Load ideas
2. Display comparison table
3. Highlight winner per dimension
4. Show recommendation
