## Trust Equation

```
Value Delivered > Value Requested = Trust
Value Requested > Value Delivered = Churn
```

## Focus Areas

**Free Tier Value** - Genuinely useful or frustratingly limited? Core goal achievable?

**Premium Clarity** - Benefits clearly communicated? Pricing transparent? Features demonstrated before purchase?

**Paywall Timing** - After value demonstration (good) or before (bad)? At high intent (optimal) or interrupting (frustrating)?

**Paywall Dismissibility** - Easy to close? X button visible? Guilt-free dismissal? No dark patterns?

**Conversion Flow** - Clear value prop, social proof, risk reversal, trial clarity, subscription terms visible

## Dark Pattern Checklist

- [ ] No hidden recurring charges
- [ ] Cancel as easy as subscribe
- [ ] No guilt-tripping copy
- [ ] No fake urgency/scarcity
- [ ] Free trial terms crystal clear
- [ ] No accidental purchases

## Output

Balance score, monetization model summary, value perception table, paywall audit, dark pattern check. Recommend: increase conversion, improve perception, adjust balance.

## Red Flags

- Paywall before any value
- Aggressive countdown timers
- "Special offer" that never expires
- Hidden cancel button
- Charging immediately after "free trial"
- Arbitrary feature restrictions
